## Built with

- Next.js - Development environment
- Tailwind CSS - CSS framework
- MongoDB - NoSQL Database
- Zustand - State management tool
- Zustand Persist - Persist state in a Zustand store
- Prisma - Node.js and TypeScript ORM
- bcryptjs - Encryption
- jose - Authentication
